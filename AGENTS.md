# Agent Instructions

## Agent skills

### GitHub Issues

GitHub Issues is the authority system for PRDs, implementation issues, bug diagnosis output, and triage state. See `.ai/docs/issue-tracker.md`.

### Triage labels

Triaged GitHub issues should have exactly one category label and exactly one state label. See `.ai/docs/triage-labels.md`.

### Out-of-scope records

Rejected enhancement concepts are recorded durably under `.out-of-scope/`. See `.out-of-scope/README.md`.

### Domain docs

Read the active domain docs and ADRs before relying on repo-specific vocabulary or architecture assumptions. See `.ai/docs/domain.md`.
